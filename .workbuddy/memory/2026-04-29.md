### 项目背景
用户想开发一款面向城市白领的个人工作助理软件，功能包括：
- 文档撰写辅助（AI）
- 表格生成（AI）
- 飞书/微信消息聚合整理（AI摘要）
- 日报/周报/月报自动生成（AI）
- 番茄钟 + 健康提醒（喝水、久坐）
- 轻松一刻（外卖入口、休闲推荐）

### 关键决策
- **技术栈**：Vue 3 (CDN) + Element Plus + Tailwind CSS + localStorage/IndexedDB
- **AI服务**：DeepSeek（免费额度够用，前端直调无CORS问题）
- **API Key管理**：必须通过设置弹窗输入，存localStorage，严禁硬编码
- **存储策略**：用户可选本地或云端同步（默认本地）
- **开发工具**：Cursor
- **消息接入**：飞书/微信官方API受限，第一版支持手动导入
- **后续版本**：桌面客户端、智能日程提取、会议纪要、知识库等8项功能列入路线图
